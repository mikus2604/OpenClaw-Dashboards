## 🗺️ What is OpenClaw?

[OpenClaw](https://github.com/openclaw/openclaw) is an open-source AI agent gateway. It runs Claude Code (and other models) as persistent, autonomous agents — chatting via Telegram/WhatsApp/Discord, running cron jobs, managing memory, spawning sub-agents, and more.

This repo lists every community-built dashboard for OpenClaw so you can see your agents, costs, tasks, and logs without hunting through terminal windows.

## 🔄 How Star Counts Stay Fresh

Star counts use **[shields.io](https://shields.io)** badges — they query the GitHub API live on every page load. No scheduled jobs needed.

Additionally, a [GitHub Actions workflow](.github/workflows/update-stars.yml) runs every Monday and commits a [`data/stars.json`](data/stars.json) snapshot. This powers historical tracking and will trigger an issue if a repo's rank changes by 2+ positions week-over-week.
